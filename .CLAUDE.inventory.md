## Key Conventions (from CLAUDE.md)

- **Personality injection**: `bot.personality` (Personality dataclass) holds bot strings + LLM endpoint — cogs read from it, not module-level constants
- **LLMClient endpoint**: Instantiated with `chat_endpoint` param per bot (e.g. `/bratchat`, `/bonniebot`)
- **Monorepo bots**: `bratbot.events.*` is shared; `botname.commands.*` is bot-specific. Set `_COG_PACKAGES = ("bratbot.events", "botname.commands")` in each bot's `bot.py`
- **Event dedup**: `MessageCog._processed_ids` (`deque(maxlen=1000)`) guards against Discord gateway replay
- **New services**: Declare type annotation on `BratBot` class AND initialize in `setup_hook`
- **`/help` rule**: Update `src/bratbot/commands/help.py` whenever any command changes (same commit)
- **Hotfix path**: SCP fails via RunPod gateway — use `curl raw.githubusercontent.com` onto pod + `supervisorctl restart`
